# 💰 Budget Tracker

A full-stack Budget Tracking Web Application to manage daily expenses, categorize spending, and visualize trends.

---

## 🚀 Features
- ➕ Add Transactions (category, amount, date)
- 📊 Dashboard with Total Amount
- 📈 Daily Spending Line Chart
- 🗂 Category-wise Summary (Total per category)
- 📝 Transaction History
- ✏️ Update Transactions
- ❌ Delete Transactions

---

## 🛠 Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Java (HttpServer)
- Database: SQLite
- Charts: Chart.js
